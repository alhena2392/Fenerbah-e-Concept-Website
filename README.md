# 💛💙 Fenerbahçe Sports Club - Fan Platform

This project is a modern and dynamic web experience designed for Fenerbahçe fans, developed as a **hobby project in just a few hours**.

### 🚀 Features
- **Match Center:** Real-time match tracking and fan voting system.
- **Squad 25/26:** Future-season squad predictions with detailed player stats and headshots.
- **Player Comparison:** Interactive module to compare star players side-by-side.
- **Digital Museum:** An interactive journey through the club's glorious history.
- **Premium Design:** Built with a "Yellow & Navy" theme using modern UI/UX principles (Glassmorphism, Tailwind CSS v4).

### 🛠️ Tech Stack
- **React 19** & **Vite**
- **Tailwind CSS v4**
- **Framer Motion** (Smooth animations)
- **Lucide React** (Icons)

### 📈 Development Progress
The site is currently in an amateur stage and was built as a proof of concept. It will continue to be developed as a hobby, with more interactive features and fan-specific zones planned for the future.

### 💻 Local Installation
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
